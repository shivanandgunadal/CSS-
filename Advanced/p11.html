CSS Pseudo-elements..!

he CSS pseudo-elements allow you to style the elements or parts of the element without adding any IDs or classes to them.

CSS3 introduced a new double-colon (::) syntax for pseudo-elements to distinguish betwwen them and pseudo-classes.

selector::pseudo-elelment { property: value; }
